# ShieldPay (Insecure Baseline for Arko Lab)

This demo is intentionally insecure for training and includes ARKO-LAB-01..09 markers.

## WARNING
Do not deploy this project to production.

## Run
```bash
cd shieldpay
npm install
npm run dev
```

Open http://127.0.0.1:8788 and login with `merchant@demo.com` / `Demo1234!`.
